# Generates the subtext of the HTML pages for non registers with a summary of the number of codechecks and number of codechecks/ venues etc.

Generates the subtext of the HTML pages for non registers with a summary
of the number of codechecks and number of codechecks/ venues etc.

## Usage

``` r
generate_html_subtext_non_register(table, filter, subcat = NULL)
```

## Arguments

- table:

  The table to showcase in the html

- filter:

  A string specifying the filter applied (e.g., "venues",
  "codecheckers").

- subcat:

  An optional string for the subcategory (if applicable).

## Value

The subtext to put under the html title
