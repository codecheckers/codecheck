# Summarise manifest files.

List manifest.

## Usage

``` r
list_manifest_files(root, metadata, check_dir)
```

## Arguments

- root:

  \- Path to the root folder of the project.

- metadata:

  \- the codecheck metadata list.

- check_dir:

  \- folder where outputs have been copied to (codecheck/outputs)

## Value

A dataframe containing one row per manifest file.
