# Function for checking ceritificate id if there is a codecheck_yaml. If there is a mismatch a stop is sent. Else a warning is thrown.

Function for checking ceritificate id if there is a codecheck_yaml. If
there is a mismatch a stop is sent. Else a warning is thrown.

## Usage

``` r
check_certificate_id(entry, codecheck_yaml)
```

## Arguments

- entry:

  The registry entry

- codecheck_yaml:

  The codecheck yaml file

## Value

None
