# Print a latex table to summarise CODECHECK metadata

Print a latex table to summarise CODECHECK metadata

## Usage

``` r
latex_summary_of_metadata(metadata)
```

## Arguments

- metadata:

  \- the codecheck metadata list.

## Value

The latex table, suitable for including in the Rmd

## Details

Format a latex table that summarises the main CODECHECK metadata,
excluding the MANIFEST.
