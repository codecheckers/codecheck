# Generates postfix hrefs for the venues/ codecheckers list pages

Generates postfix hrefs for the venues/ codecheckers list pages

## Usage

``` r
generate_html_postfix_hrefs_non_reg(filter, table_details)
```

## Arguments

- filter:

  The filter being used such as "venues" or "codecheckers"

- table_details:

  A list of metadata about the table (e.g., title, subtext, extra text).

## Value

A list of the hrefs.
