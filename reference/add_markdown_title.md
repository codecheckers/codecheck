# Function to add the markdown title based on the specific register table name.

Function to add the markdown title based on the specific register table
name.

## Usage

``` r
add_markdown_title(table_details, md_table, filter)
```

## Arguments

- table_details:

  List containing details such as the table name, subcat name.

- md_table:

  The markdown table where the title needs to be added

- filter:

  The filter

## Value

The modified markdown table
