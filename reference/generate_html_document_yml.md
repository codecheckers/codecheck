# Dynamically generates a html_document.yml with the full paths to the index header, prefix and postfix.html files.

Dynamically generates a html_document.yml with the full paths to the
index header, prefix and postfix.html files.

## Usage

``` r
generate_html_document_yml(output_dir)
```

## Arguments

- output_dir:

  The output directory where the html_document needs to be saved.
