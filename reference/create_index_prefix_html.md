# Dynamically generates the index_prefix.html with navigation header and breadcrumbs

Dynamically generates the index_prefix.html with navigation header and
breadcrumbs

## Usage

``` r
create_index_prefix_html(output_dir, filter = NA, table_details = list())
```

## Arguments

- output_dir:

  The output directory

- filter:

  The filter name

- table_details:

  List containing details such as the table name, subcat name.
