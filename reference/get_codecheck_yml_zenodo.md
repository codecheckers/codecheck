# Retrieve a codecheck.yml file from a Zenodo record

Retrieve a codecheck.yml file from a Zenodo record

## Usage

``` r
get_codecheck_yml_zenodo(x, sandbox = FALSE)
```

## Arguments

- x:

  the record ID on Zenodo

- sandbox:

  connect with the Zenodo Sandbox instead of the real service
