# Function for adding clickable links to the issue number of each report in the register table.

Function for adding clickable links to the issue number of each report
in the register table.

## Usage

``` r
add_issue_number_links(register_table, register)
```

## Arguments

- register_table:

  The register table

- register:

  The register from the register.csv file

## Value

The adjusted register table
