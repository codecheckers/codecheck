# Generates the hrefs to set in the postfix.html file for the rendering of register tables.

Generates the hrefs to set in the postfix.html file for the rendering of
register tables.

## Usage

``` r
generate_html_postfix_hrefs_reg(filter, table_details)
```

## Arguments

- filter:

  The filter name

- table_details:

  List containing details such as the table name, subcat name.

## Value

A list with the hrefs
